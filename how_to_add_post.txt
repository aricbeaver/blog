1) Create new post in blog/_posts/<name_of_post>.md
2) Build jekyll using command "jekyll build"
3) Copy tags directory generated by jekyll to root directory because GitHub Pages will not generate this automatically
	"rm -R _site"
	"rm -R tag"
	"cp -R _site/tag/ tag/"
4) Serve jekyll locally to review post "jekyll serve"
5) Add and commit files to GitHub
	"git add ."
	"git status"
	"git commit -m "message""
	"git push"
